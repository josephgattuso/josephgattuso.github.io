About

Hi there! My name is Joseph and I'm an aspiring Web Developer based in London. My current focus is on building gorgeous web applications, mostly with code based on JavaScript frameworks.

One of the main reasons for my application to Founders and Coders is to learn how to code! I appreciate how the coursework is structured through pair-programming and completing projects (I am very much a tactile learner!)

It is true there are some great resources freely available to learn programming. But we need to meet new friends and colleagues who share our visions to make greater change in the world today. This is why I am applying to Founders and Coders; to become part of a fantastic developer community.
